Overview

This project investigates various classical lighting aspects in computer graphics based on an OpenGL tutorial. We use three.js as a base and define a custom material using shaderMaterial. This construct copies various attributes and uniforms to the GPU for us and lets us specify additional data and shader code. The project includes three.js camera, geometry and skybox with everything else manually defined in the vertex and fragment shaders using GLSL. Features include Phong lighting, texturing, normal mapping, specular mapping, environment mapping, reflection, refraction and shadow mapping using an orthographic camera and a texture buffer. The texture, normal map and specular map are edited versions of images from Share Textures and the skybox image is taken from HDRI Haven.
