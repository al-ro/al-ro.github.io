Signed Distance Fields

Part of my research is concerned with generating signed distance fields from STL files. The SDF is used to describe complex geometries in regular Cartesian grids and generate cut cells at the object boundaries. The main aims were increased generation speeds and improved robustness compared to existing methodologies. An near-exact SDF field can be constructed from closed orientable STL geometries where the precision is limited only by the resolution of the target computational mesh.

(Left: A narrow band SDF is generated at high resolution. Right: The 0 level set (white) matches the STL input slice (purple) to within Δx.)

Robustness

As industrial geometries can feature complex topology where the surfaces are far from smooth, the produced code detects and deals with cases of high curvature at vertices.

(High curvature geometries and correct SDFs)

Performance

A CUDA implementation was produced which reads in an STL file, quickly builds an internal geometry representation from unordered surface features and then generates an SDF around the boundary to a user defined maximum distance. Based on a testing set of STL files ranging from 52,000 to 2,500,000 faces, the resulting code pipeline from reading in the file to handing over to a CFD solver mostly takes on the order of seconds for low distance bands.
