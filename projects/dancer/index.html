Overview

Three.js offers an easy interface to create 3D WebGL scenes with lights, different materials and STL import using STLLoader.js. The dancer mesh is a simplified version of Ballet Dancer 03 by maitan. The flakes move according to a noise field and a rotation field around the z-axis.
