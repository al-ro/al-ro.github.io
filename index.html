About

I am a PhD student at the University of Cambridge. My research investigates speeding up CFD simulations and signed distance field generation using multi-GPU parallelisation for regular rectilinear grids with embedded cut cell geometries.

This page features several coding projects based on concepts which I have found interesting or challenging. Most of these are based on tutorials and code examples found online. The page includes simulations, graphics and other software applications with the intent of documenting some of my side projects.
